# Phosphor

A static documentation site generator with the Terminal Noir theme.
Write Markdown, get a polished dark-mode docs site with search, navigation, and rich components.

## Requirements

- Python 3.8+
- PyYAML (`pip install pyyaml`)

That's it. No Node.js, no bundlers, no JavaScript toolchain.

## Quick Start

```bash
git clone https://github.com/gbasran/phosphor-docs ~/phosphor-docs
cd ~/phosphor-docs && pip install pyyaml && ./install.sh

# Use from any project directory
cd ~/my-project
phosphor init
phosphor build
phosphor serve
```
