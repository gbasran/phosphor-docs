### How It Works

:::pipeline
Write Markdown -> Configure YAML -> Build Site -> Deploy
:::

### Quick Start

```terminal
$ phosphor init my-docs
$ cd my-docs
$ phosphor build
Site built to _site/
$ phosphor serve
Serving at http://localhost:8000
```

:::tip Get started quickly
Run `phosphor init` in any directory to scaffold a new docs project with example content.
:::
